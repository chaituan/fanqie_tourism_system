# 番茄三级分销旅游系统

#### Description
本系统采用Codeigniter框架开发的一款带分销功能的旅游系统，主要模块有游记分享，驴友自由开团，精选相册，多分类搜索，多级分销返利，领队申请，积分兑换，资讯模块，微信支付，活动订单。此版本为公众号版本。
